3-668-ఉ.
  • ఉపకరణాలు:

  •  
  •  

"అంచిత దివ్యమూర్తి! పరమాత్మక! యీ కలుషాత్ముఁ డైన న
క్తంచరుఁ డస్మదీయ వరగర్వమునన్ భువనంబు లెల్లఁ గా
రించు మదించి యిట్టి విపరీతచరిత్రునిఁ ద్రుంప కిట్లుపే
క్షించుటగాదు వీని బలిసేయు వసుంధరకున్ శుభంబగున్.

టీకా:

అంచిత = చక్కటి; దివ్య = దివ్యమైన; మూర్తిన్ = స్వరూపా; పరమాత్మక = నారాయణ {పరమాత్మక - పరమ (అత్యన్నతమైన) ఆత్మకలవాడ, విష్ణువు}; ఈ = ఈ; కలుషాత్ముడు = పాపాత్ముడు; ఐన = అయిన; నక్తం = రాత్రులందు; చరుడు = తిరుగువాడు; అస్మదీయ = నా యొక్క; వర = వరము వలని; గర్వమునన్ = గర్వముతో; భువనంబులున్ = లోకములు; ఎల్లన్ = సమస్తమును; గారించు = చీకాకుపెట్టు; మదించి = గర్వించి; ఇట్టి = ఇటువంటి; విపరీత = చెడు; చరిత్రునిన్ = వర్తనకలవానిన; త్రుంపకన్ = సంహరించకుండగ; ఇట్లు = ఈ విధముగ; ఉపేక్షించుట = నిర్లక్ష్యము చేయుట; కాదు = సరికాదు; వీనిన్ = వీడిని; బలి = బలిచ్చుట; చేయు = చేయుము; వసుంధర = భూదేవి; కున్ = కి; శుభంబున్ = శుభము; అగున్ = అగును.

భావము:

“దివ్యస్వరూపా! పరమాత్మా! ఈ పాపాత్ముడైన రాక్షసుడు నా వరంవల్ల గర్వించి లోకాలనన్నిటినీ చీకాకు పరుస్తున్నాడు. ఇటువంటి దుశ్చరిత్రుని చంపకుండా ఇలా నిర్లక్ష్యం చేయడం సరి కాదు. వీనిని సంహరించు. భూదేవికి శుభం కలుగుతుంది.
