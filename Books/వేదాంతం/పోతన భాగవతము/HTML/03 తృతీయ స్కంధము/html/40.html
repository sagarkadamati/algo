3-762-వ.
  • ఉపకరణాలు:
  •  
  •  
  •  

అంత స్వాయంభువుండు గనకరథారూఢుం డగుచు నిజభార్యా సమేతుం డై భర్తృవాంఛాపరయైన పుత్రికం దోడ్కొనుచు నిజ తనూజకుం దగిన వరు నన్వేషించుచు భువనంబునం గలయం గ్రుమ్మరి; యెందునుం గానక వచ్చివచ్చి.

టీకా:

అంతన్ = అప్పుడు; స్వాయంభువుండు = స్వాయంభువుడు; కనక = బంగారు; రథా = రథమును; ఆరూఢుండు = ఎక్కినవాడు; అగుచున్ = అవుతూ; నిజ = తన యొక్క; భార్యా = భార్యతో; సమేతుండు = కూడినవాడు; ఐ = అయ్యి; భర్తృ = భర్తను; వాంఛా = కోరుట యందు; పర = లగ్నమైనది; ఐన = అయిన; పుత్రికన్ = కూతురిని; తోడ్కొనుచున్ = కూడా తీసుకెళుతూ; నిజ = తన యొక్క; తనూజ = కూతురున; కున్ = కు; తగిన = తగిన; వరున్ = భర్తను; అన్వేషించుచు = వెతుకుతూ; భువనంబుననన్ = లోకములో; కలయన్ = కలియ; గ్రుమ్మరి = తిరిగి; ఎందునున్ = ఎక్కడను; కానక = కనిపించక; వచ్చివచ్చి = వెళ్ళివెళ్లి.

భావము:

అప్పుడు స్వాయంభువ మనువు బంగారు రథమెక్కి భార్యతో కూడి పెండ్లికి ఎదిగిన తన కుమార్తెను వెంట బెట్టుకొని, ఆమెకు తగిన వరుణ్ణి అన్వేషిస్తూ లోకాలన్నీ తిరిగి, ఎక్కడా అటువంటివాడు కనబడక వచ్చి వచ్చి…

నంద మెలర్పఁ జూచి నయనంబుల రాలిన బాష్పముల్ ధరన్

భావము:
