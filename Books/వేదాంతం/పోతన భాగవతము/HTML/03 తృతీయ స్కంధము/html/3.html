3-36-వ.
  • ఉపకరణాలు:
  •  
  •  
  •  

అని యిట్లు దుర్యోధనుం డాడిన దురాలాపంబులు దనకు మనస్తాపంబు సేయం గార్యంబు విచారించి ధైర్యం బవలంబించి యొండు వలుకనొల్లక శరశరాసనంబులు విడిచి క్రోధంబు నడంచి విదురుండు వనంబునకుం జని, యందు.

టీకా:

అని = అని; ఇట్లు = ఈవిధముగ; దుర్యోధనుండు = దుర్యోధనుడు; ఆడిన = పలికిన; దురాలాపంబులు = దుడుకుమాటలు; తన = తన; కున్ = కు; మనస్ = మనసునకు; తాపంబున్ = బాధ; చేయన్ = కలుగజేయగా; కార్యంబున్ = చేయవలసిన పని; విచారించి = ఆలోచించుకొని; ధైర్యంబున్ = ధైర్యమును; అవలంభించి = వహించి; యొండు = బదులు; పలుకన్ = పలుకుటకు; ఒల్లక = ఒప్పుకొనక; శర = బాణము; శరాసనంబులు = విల్లులను; విడిచి = వదిలేసి; క్రోధంబున్ = కోపమును; అడంచి = అణచుకొని; విదురుండు = విదురుడు; వనంబున్ = అడవి; కున్ = కి; జని = వెళ్ళి; అందున్ = దానిలో.

భావము:

ఇలా పలికిన దుర్యోధనుడి దుడుకు మాటలకు విదురుని మనస్సు కుతకుత ఉడికిపోయింది. అయినా విదురుడు కర్తవ్యం గుర్తుకు తెచ్చుకొని, గుండె నిబ్బరించుకొని, ఇంకేం మాట్లాడకుండా, ధనుర్బాణాలు విడిచిపెట్టి, ఆగ్రహాన్ని అణచిపెట్టి అరణ్యానికి వెళ్ళిపోయాడు.
