"నరేంద్రా! నిన్ను నారోపితశరశరాసను సర్వప్రదేశంబు లందును విలోకించుచున్నవాడ; నే నెక్కడ నుండుదు నానతిమ్మ" నిన, రాజన్యశేఖరుండు ప్రాణవధ స్త్రీ ద్యూత పానంబు లనియెడు నాలుగు స్థానంబుల నిచ్చి, మఱియు నడిగిన సువర్ణ మూలం బగు నసత్య మదకామ హింసా వైరంబు లనియెడు పంచప్రదేశంబుల నొసంగి యితరస్థలంబుల స్పృశియింపకుండ నియమించె; నిట్లు కలినిగ్రహంబుసేసి హీనంబు లయిన తప శ్శౌచ దయ లనియెడి మూఁడుపాదంబులు వృషభమూర్తి యయిన ధర్మదేవుని కిచ్చి విశ్వంభరకు నిర్భరం బయిన సంతోషంబు సంపాదించి.

నరేంద్రా = రాజా {నరేంద్ర - నరులకు ఇంద్రుడు రాజు}; నిన్నున్ = నిన్ను; ఆరోపిత = ఎక్కుపెట్టిన; శరశరాసను = విల్లంబులు కలవానిగా; సర్వ = సమస్త; ప్రదేశంబులు = స్థలములు; అందునున్ = లోను; విలోకించుచున్ = చూచుచు; ఉన్నవాడ = ఉన్నాడను; నేన్ = నేను; ఎక్కడన్ = ఎక్కడ; ఉండుదున్ = ఉండను; ఆనతి = ఆజ్ఞ; ఇమ్ము = ఇమ్ము; అనిన = అనగా; రాజన్య = రాజులలో, క్షత్రియులలో; శేఖరుండు = శిఖరము వంటివాడు, శ్రేష్ఠుడు; ప్రాణవధ = జీవహింసయు; స్త్రీ = స్త్రీలోలత్వమును; ద్యూత = ద్యూతమును, జూదమును; పానంబులు = మద్యపానమును, తాగుడును; అనియెడు = అనెడు; నాలుగు = నాలుగు (4); స్థానంబులన్ = స్థలములను; చోటులను; ఇచ్చి = ఇచ్చి; మఱియున్ = ఇంకను; అడిగిన = అడుగగా; సువర్ణ = బంగారము, విత్తము; మూలంబు = మూలకారణము; అగు = అయిన; అసత్య = అసత్యమును, అబద్దమును; మద = మదమును, గర్వమును; కామ = కామమును; హింస = హింసయు; వైరంబులు = వైరములు, కలహములు; అనియెడు = అనే; పంచ = ఐదు; ప్రదేశంబులన్ = స్థలములను; ఒసంగి = ఇచ్చి; ఇతర = ఇతర; స్థలంబులన్ = స్థలములను; స్పృశియింపకన్ = ముట్టుకొనక; ఉండన్ = ఉండునట్లు; నియమించెన్ = కట్టడి చేసెను; ఇట్లు = ఈ విధముగ; కలి = కలి యొక్క; నిగ్రహంబు = కట్టడి; చేసి = చేసి; హీనంబులు = క్షీణించినవి; అయిన = అయినట్టి; తపస్ = తపస్సును; శ్శౌచ = శౌచమును; దయలు = దయయు; అనియెడి = అనెడి; మూఁడు = మూడు (3); పాదంబులున్ = కాళ్ళు; వృషభ = ఎద్దు; మూర్తి = రూపమున ఉన్నవాడు; అయిన = అయినట్టి; ధర్మదేవుడు = ధర్మదేవత; కిన్ = కు; ఇచ్చి = ఇచ్చి; విశ్వంభర = విశ్వమును భరించునది, భూదేవి; కున్ = కి; నిర్భరంబు = నిర్భరము, అధికము; అయిన = అయినట్టి; సంతోషంబున్ = సంతోషమును; సంపాదించి = కలిగించి.

భావము:

రాజేంద్రా! ఎటు చూస్తే అటు సర్వత్రా ధనుర్ధరుడవైన నీ రూపమే నా లోచనాలకు గోచరిస్తున్నది. మరి నేను ఎచట తలదాచుకొనేది ఆజ్ఞాపించు” అని అలమటించే కలిపురుషుని ప్రార్థన ఆలకించి పరీక్షిత్తు వానికి ”జూదం, మద్యపానం, స్త్రీలు, ప్రాణివధ” అనే నాలుగు స్థానాలు ఇచ్చాడు. ఇంకా అతడు అర్థించగా మహారాజు ”సువర్ణం” అనే అయిదో స్థానాన్ని ఇచ్చాడు. ఇంకా అతడు అర్థించగా మహారాజు ”సువర్ణం” అనే అయిదో స్థానాన్ని కూడా అనుగ్రహించాడు. అందువల్ల అసత్యం, గర్వం, కామం, హింస, వైరం అనే ఇంకా ఐదు స్థానాలు కూడా కలి నివసించే ప్రదేశాలు ఐనాయి. ఈ స్థానాలలో తప్ప ఇతర స్థలాలను స్పృశించవద్దని కలిని కట్టడి చేసాడు పరీక్షిత్తు. ఈ ప్రకారం ఆ మహారాజు కలిపురుషుణ్ణి నిగ్రహించి పోగొట్టుకొన్న తపస్సు, శౌచం, దయ అనే మూడు పాదాలూ వృషభమూర్తి అయిన ధర్మదేవతకు అర్పించి భూదేవికి అపారమైన ఆనందాన్ని కలిగించాడు.

గజనామధేయపురమున
గజరిపుపీఠమున ఘనుఁడు గలిదమనుం డున్
గజవైరిపరాక్రముఁ డే
గజిబిజి లేకుండఁ దాల్చెఁ గౌరవలక్ష్మిన్.

టీకా:

గజనామధేయపురమున = హస్తినాపురమునందు {గజ నామధేయ పురము - గజ (హస్తిన అను) నామధేయ (పేరు కల) పురము (పురము) హస్తినాపురము}; గజరిపుపీఠమున = సింహాసనమున {గజ రిపు పీఠము - గజ (ఏనుగు) రిపు (శత్రువు, సింహము) పీఠము (ఆసనము), సంహాసనము}; ఘనుఁడున్ = గొప్పవాడు; కలి = కలిని; దమనుండున్ = అణచిన వాడును; గజ వైరి పరాక్రముడు = సింహపరాక్రముడు {గజ వైరి పరాక్రముడు – (గజ (ఏనుగు) వైరి (శత్రువు, సింహ) పరాక్రముఁడు (పరాక్రమము కలవాడు, - సింహపరాక్రముడు}; ఏ = ఎట్టి; గజిబిజి = గజిబిజి, గందరగోళం; లేకుండన్ = లేకుండగ; తాల్చెన్ = ధరించెను; గౌ(కౌ)రవ = గౌరవము అను, కౌరవ సామ్రాజ్యం అను; లక్ష్మిన్ = సంపదను.

భావము:

సింహపరాక్రముడు, కలిని నిగ్రహించిన ఘనుడు ఆయిన పరీక్షిత్తు హస్తినాపురంలో సింహాసనాసీను డై కౌరవ రాజ్యలక్ష్మిని గౌరవపూర్వకంగా ప్రశాంతంగా పరిపాలించాడు.

ఇట్లు కృష్ణుని యనుగ్రహంబున నశ్వత్థామ బాణపావకంబు వలన బ్రతికి; వృషభమూర్తి యయిన ధర్మదేవతకు నభయం బిచ్చిన పరీక్షిన్నరేంద్రుండు బ్రాహ్మణశాపప్రాప్త తక్షకాహిభయంబు వలనఁ బ్రాణంబులు వోవు నని యెఱింగి, సర్వసంగంబులు వర్జించి శుకునకు శిష్యుండై విజ్ఞానంబు గలిగి, గంగాతరంగిణీ తీరంబునం గళేబరంబువిడిచె వినుఁడు.

"అయ్యా! పెద్దలైన మీ వంటి వారు కావాలని కోరగా కమలాక్షుని గాథలు వినిపించే మహాభాగ్యం నాకు ప్రాప్తించింది. నా జన్మం ధన్యమైంది. నా జీవితం సజ్జన సమ్మాన్య మయింది.
